Formulário de Interação

Este projeto contém uma página HTML simples com um formulário de coleta de informações pessoais e preferências de trabalho.

Estrutura do Projeto

index.html → Página principal contendo o formulário.

style.css → (referenciado no HTML, mas não incluído) arquivo para personalização visual do formulário.

Funcionalidades do Formulário

Campos de texto para dados pessoais:

Nome

Endereço

Bairro

Cidade

Área de mensagem:

Caixa de texto com até 500 caracteres.

Preferências de trabalho:

Tipo de modelo: Presencial, Remoto ou Híbrido.

Período de trabalho: Manhã, Tarde ou Noite.

Seleção de estado:

Lista suspensa com todas as unidades federativas do Brasil.

Aceite dos termos de uso:

Checkbox para confirmação.

Botões de envio e limpeza de campos:

“Enviar dados”

“Limpar os campos”
